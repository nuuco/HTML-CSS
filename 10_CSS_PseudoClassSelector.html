가상 클래스 선택자1

first-child, last-child, nth-child(n)

찾는 순서 :
  1. 가상 클래스가 붙기 전 선택 요소 파악
  2. 해당 요소들의 형제 요소 파악
  3. 형제 요소 내에서 몇번째 요소인지 파악
  4. 내가 쓴 nth-child와 n번째가 일치하는 확인
주의!) "선택된 애들" 중에 n번째가 아니다! "형제 요소" 중에 n번째다!
  • first-child
  • nth-child(2)
  • nth-child(3)
  • last-child

cf ) 선택된 요소에 첫번째 요소가 없으면?

first-child가 아예 적용되지 않는다.

last-child와 nth-child 도 해당 형제 요소가 선택되지 않았으면 아예 적용되지 않는다.

  • first-child 적용한거임.
  • nth-child(2)
  • nth-child(3)
  • last-child

cf ) nth-child(n)

()안에 n 을 매개변수로 활용해 함수를 넣을 수 있다.

nth-child(2n) : 짝수만 선택 (even) / (2n-1) : 홀수만(odd)

first-of-type, last-of-type, nth-of-type(n)

찾는 순서 :
  1. 가상 클래스가 붙기 전 선택 요소 파악
  2. 해당 요소들의 형제 요소 파악
  3. 형제 요소 중 같은 태그(type) 내에서 몇번째 요소인지 파악
  4. 내가 쓴 nth-of-type과 n번째가 일치하는 확인
주의!) "선택된 애들" 중에 n번째가 아니다! "형제 요소" 중 "같은 type(태그)" 중에 n번째다!
  • first-of-type
  • nth-of-type(2)
  • nth-of-type(3)
  • last-of-type

cf ) 선택된 요소에 첫번째 요소가 없으면?

first-of-type가 아예 적용되지 않는다.

last-of-type와 nth-of-type 도 해당 형제 요소가 선택되지 않았으면 아예 적용되지 않는다.

cf ) 만약 선택된 요소들의 선택되지 않은 형제 요소에 다른 태그(type)가 섞여있다면?

다른 태그 형제들은 무시하고 해당 태그(type) 중에서 첫번째 자식들이 선택된다.

    div 태그 1
  • li 태그 1
    • div 태그 2
  • li 태그 2

cf ) 만약 선택된 요소들 중에 여러 태그(type)가 섞여있다면?

각 태그의 형제들 중에서 첫번째 자식들이 선택된다.

    div 태그 1
  • li 태그 1
    • div 태그 2
  • li 태그 2
