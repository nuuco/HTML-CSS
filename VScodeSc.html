VScode 단축키

에디터 창 제어

현재 창 닫기   Ctrl + w
닫은 창 다시 열기   Ctrl + Shift + t

사이드바 토글   Ctrl + b
사이드바 - 탐색기   Ctrl + Shift + e (explore)
사이드바 - 전체 검색   Ctrl + Shift + f (find)

에디터 확대   Ctrl + '+'('=')
에디터 축소   Ctrl + '-'

소스코드 편집

들여쓰기   Tab (커서 맨 앞에) 혹은 Ctrl + ] (커서 위치 상관X)
내어쓰기   Shift + Tab (커서 맨 앞에) 혹은 Ctrl + [ (커서 위치 상관X)

아래에 행 삽입   Ctrl + Enter (커서 위치 상관X)
위에 행 삽입   Ctrl + Shift + Enter (커서 위치 상관X)

현재 행 이동   Alt + 위아래 방향키
현재 행 복사   Alt + Shift + 위아래 방향키
현재 행 삭제   Ctrl + Shift + k
현재 행 잘라내기   Ctrl + x

주석토글   Ctrl + /
