HTML Emmet

    1. 자동완성 : tab키   → div(tab키)
    2. 텍스트 : {}   → div{안녕}
    3. 자식(하위)요소 : >   → div>div
    4. 형제 요소 : +   → div>p+a
    5. 올라가기 : ^   → div>p^div
        
        ^ → 한 수준 위로 이동
        ^^ → 두 수준 위로 이동
        
    6. 반복하기 : *   → ul>li*4
    7. 그룹화 : ()   → ul>(li>a)*4
    8. 클래스 : .   → .class-name
    9. 아이디 : #   → #id
    10. 속성: [attr]   → img[alt="이미지 설명"]
    11. 넘버링 : $   → .item$*6    = 1 ~
        
        $$   → 01 처럼 두자리 숫자로 표시된다.
        
    12. 텍스트 넘버리: {$}   → .item{번호$}
    13. 동일한 텍스트가 있는 요소 반복   → section>h3{12개의 열}+div*12{1칸 크기}
        
        ※ 텍스트는 *개수 뒤에 써도 되고, 앞에 써도 된다.
