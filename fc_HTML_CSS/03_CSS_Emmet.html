CSS Emmet

cheat-sheet 사이트

많이 쓰는 Emmet 예제

    - mt10     ➡ margin-top: 10px;
    - pb10     ➡ padding-bottom: 10px;
    - w100     ➡ width: 100px;
    - h100p    ➡ height: 100%;
    - bd       ➡ border: 1px solid #000;
    - bgc      ➡ background-color: #fff;
    - fsz10    ➡ font-size: 10px;
    - c#ddd    ➡ color: #ddd;
    - z10      ➡ z-index: 10;
