02. 메타데이터 요소

1. title 태그는 안 쪽에 요소가 있어도 무시. 무조건 문자열만 쓰기.

2. 하나의 문서 안에 하나의 head, 하나의 head 안에 하나의 title

3. title 은 SEO 에 큰 영향을 준다. - 키워드 뭉치 제목X 문장이나 설명으로 쓰기

4. mata charset 은 head의 첫번째 정보로 넣거나, title 태그 전에 넣어주길 권장

5. MIME 타입(밈 타입) : 외부파일의 타입을 알려주는 것 작성법. MIME MDN

6. script 태그는 body 태그의 맨 마지막에 적어주는걸 권장!
   Why? 자바스크립트 문서를 해선하느라 화면 랜더링이 중단되는 걸 막기위해서.
